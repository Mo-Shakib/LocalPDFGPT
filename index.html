Why LocalPDFGPT?

Skip the API fees and privacy concerns. Run a state-of-the-art RAG pipeline completely on your own hardware.

100% Private & Offline

Your documents never leave your machine. No cloud uploads, no data harvesting. Perfect for sensitive documents.

Local Llama 3.2 Power

Leverages Ollama to run the latest Llama 3.2 models efficiently on consumer hardware.

Dual Interfaces

Get Started in Minutes

Prerequisite: Python 3.10+ and Ollama installed.

user@local:~/projects/LocalPDFGPT

# 1. Clone the repository

$ git clone https://github.com/Mo-Shakib/LocalPDFGPT.git

$ cd LocalPDFGPT

# 2. Install dependencies

$ pip install -r requirements.txt

# 3. Pull the Llama 3.2 model

$ ollama pull llama3.2:latest

# 4. Start the application

$ python start.py

> [1] Use Command Line Interface

> [2] Start Web Interface (http://127.0.0.1:8080)

> [3] Load and Index PDF
